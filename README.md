# 🧪 VGNet Lab Control App (Flutter)

## Integrated IoT & Lab Management System

This repository contains the **full Flutter application for VGNet Lab**, designed to control and monitor **multiple devices, sensors, and systems** within the lab environment.

This app is an **extension of the VisionLock (door security system)** and acts as a **centralized control interface** for broader lab automation.

⚠️ **Completed but Not Actively Maintained**

* The application is **fully functional**

---

## 📖 About This Project

The VGNet Lab App is a **smart control system** built to manage:

* 🔐 Door access (VisionLock integration)
* 💡 Lab lighting systems
* 🌡️ Sensors and environmental monitoring
* 🧩 Multiple connected devices

It provides a **unified mobile interface** for interacting with IoT-based lab infrastructure.

---

## 🚀 Features

* 📱 Centralized dashboard for device control  
* 🔐 Integration with door security system (VisionLock)  
* 💡 Control of lights and electrical systems  
* 🌡️ Sensor data monitoring (based on setup)  
* ⚙️ Scalable architecture for adding new devices  
* 🔄 Real-time interaction with connected hardware
